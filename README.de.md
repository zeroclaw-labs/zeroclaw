### Warum Teams ZeroClaw wΟΛhlen

- **Leichtgewichtig standardmΟΛΟüig:** kleines Rust-Binary, schneller Start, geringer Speicherbedarf.

- **Kein Provider-Lock-in:** OpenAI-kompatible Provider-UnterstΟΦtzung + steckbare Custom-Endpoints.

## Schnellstart

### Option 1: Automatisierte Einrichtung (empfohlen)

Das `bootstrap.sh`-Skript installiert Rust, klont ZeroClaw, kompiliert es und richtet deine anfΟΛngliche Entwicklungsumgebung ein:


Dies wird:

1. Rust installieren (falls nicht vorhanden)
2. Das ZeroClaw-Repository klonen
3. ZeroClaw im Release-Modus kompilieren
4. `zeroclaw` in `~/.cargo/bin/` installieren

Nach dem Bootstrap lade deine Shell neu oder fΟΦhre `source ~/.cargo/env` aus, um den `zeroclaw`-Befehl global zu verwenden.

## Befehle
