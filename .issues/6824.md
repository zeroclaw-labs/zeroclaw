# [Tracker]: TUI Agent Chat

TUI Agent Chat — the interactive chat interface for conversing with ZeroClaw agents in the terminal. Renders streaming agent responses, tool calls, tool results, and approval prompts in a Ratatui-based UI.

### Current state

The TUI crate (`crates/zeroclaw-tui`) currently only contains the onboarding UI (`RatatuiUi`). No chat interface exists yet.

### Scope

- [ ] Chat view layout: message history area, input area, status bar
- [ ] Streaming text rendering: display `AgentMessageChunk` notifications as they arrive (character-by-character or chunk-by-chunk)
- [ ] Thinking/reasoning display: render `AgentThoughtChunk` in a visually distinct style (dimmed, italic, collapsible)
- [ ] Tool call display: show tool name + arguments when `ToolCall` notification arrives
- [ ] Tool result display: show tool output when `ToolResult` notification arrives
- [ ] Tool approval prompts: when `ApprovalRequest` arrives, show a prompt with approve/reject keybindings
- [ ] Input area: multi-line text input with submit (Enter or Ctrl+Enter), history (up/down)
- [ ] Cancel in-flight turn: keybinding (Ctrl+C or Esc) sends `session/cancel`
- [ ] Session management: create new sessions, switch between sessions, rename sessions
- [ ] Markdown rendering in agent responses (code blocks, bold, lists)
- [ ] Scroll back through message history
- [ ] Copy text from agent responses to clipboard
