# raw/artifacts

사진·스캔·기사 클리핑·외부 문서의 로컬 사본을 이곳에 둔다.

- 파일명 규칙: `YYYY-MM-DD-slug.{ext}` (예: `2026-04-18-photo-beach.jpg`)
- 원본 링크(URL)가 있다면 같은 이름의 `.md` 짝을 만들어 출처·캡션을 기록한다.
  - 예: `2026-04-18-article-xyz.md` + `2026-04-18-article-xyz.pdf`
- 바이너리가 커질 수 있으므로 저장소 크기를 주기적으로 점검.

LLM은 여기를 **읽기 전용**으로 취급한다.
