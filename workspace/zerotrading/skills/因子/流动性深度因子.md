# 流动性与市场深度因子

## 核心目标
监控盘口流动性，避免在衰竭市场中开大仓，减少滑点损耗。

## 监控指标
- 买一卖一价差(spread/mid): 
  - `< 0.02%` → 优质流动性，正常 size
  - `0.02%~0.1%` → 一般流动性，size 下调 30%
  - `> 0.1%` → 流动性差，建议观望
- 24h 成交量 vs 7d 均值:
  - `> 150%` → 活跃市场
  - `< 50%` → 市场萎缩，谨慎

## 触发条件
- Spread > 0.1% → 降低 size 或 HOLD
- 成交量 < 7d 均值 30% → 禁止大仓开仓（size 上限 5%）

## 输出映射
- 优质流动性 → 策略 size 不变
- 一般流动性 → size × 0.7
- 流动性不足 → size 上限 5% 或 `{"decision": "HOLD"}`

## 风险约束
- 限价单优先；市价单仅在流动性优质时使用
