# 资金费率因子

## 核心目标
通过永续合约资金费率识别市场多空情绪极端值，辅助判断趋势可持续性。

## 监控指标
- 资金费率（8h）: 
  - `> +0.05%` → 多头过热，空头信号增强
  - `< -0.05%` → 空头过热，多头信号增强
  - `[-0.01%, +0.01%]` → 中性，不影响信号
- 资金费率连续方向（3期）:
  - 连续 3 期正向 → 强多头情绪
  - 连续 3 期负向 → 强空头情绪

## 触发条件
- 费率 > 0.1% AND 持多仓 → 警告过热，减仓建议
- 费率 < -0.1% AND 持空仓 → 警告过热，减仓建议
- 极端费率（>0.3% 或 <-0.3%）→ 市场反转风险极高，发出 `CAUTION` 标记

## 输出映射
- 费率中性 → 不干预策略信号
- 费率偏多 (+0.05%~0.1%) → size 上调 10%
- 费率极热 (>0.1%) → 策略 BUY size 下调 20%（过热折扣）
- 费率极冷 (<-0.1%) → 策略 SELL size 下调 20%

## 风险约束
- 本因子为辅助信号，不单独触发开平仓
- 仅在策略产生信号时修正 size，不产生独立决策
