# 最大回撤强制止损

## 核心目标
硬性约束最大单次回撤，防止账户穿仓。

## 监控指标
- 当日已实现亏损: 超过账户净值 **3%** → 立即停止开新仓
- 浮动亏损(单笔): 超过开仓本金 **5%** → 强制平仓
- 连续亏损次数: ≥ 3 次 → 进入冷却期（暂停交易 4 小时）

## 触发条件
- 当日累计亏损 > 3% → `{"decision": "HOLD", "size": 0}` + 通知人工
- 单笔浮亏 > 5% → `{"decision": "CLOSE", "size": 0}`
- 最大回撤峰值 > 8% → `{"decision": "REQUEST_APPROVAL", "size": 0}`

## 输出映射
- 正常交易状态 → 不干预策略信号
- 触发日亏损上限 → `{"decision": "HOLD", "size": 0, "reason": "daily_loss_limit"}`
- 触发单仓止损 → `{"decision": "CLOSE", "size": 0, "reason": "position_stop_loss"}`

## 风险约束
- ⚠️ **此约束为最高优先级，不可被任何策略信号覆盖**
- 所有其他技能的 BUY/SELL 信号必须先通过本约束检查
- 冷却期内仅允许平仓操作，禁止新开仓
