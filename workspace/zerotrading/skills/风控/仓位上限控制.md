# 仓位上限与杠杆控制

## 核心目标
限制任意时刻的总敞口，防止杠杆失控放大亏损。

## 监控指标
- 单一资产最大仓位: 账户净值的 **20%**
- 总持仓名义价值: 不超过账户净值的 **150%**（相当于 1.5x 杠杆）
- 合约杠杆倍数: 默认 **5x**，极端行情下自动降至 2x

## 触发条件
- 请求开仓 size > 20% → 截断至 20%
- 总仓位名义价值 > 150% → 拒绝新增开仓
- 波动率（1h）> 8% → 杠杆自动降至 2x

## 输出映射
- 正常请求 → 透传策略信号，只修正 size 上限
- 超限请求 → `{"decision": "HOLD", "size": 0, "reason": "position_limit_exceeded"}`

## 风险约束
- size 字段最大值为 0.20（20% 净值）
- 返回的所有 BUY/SELL size 必须经过此模块截断
