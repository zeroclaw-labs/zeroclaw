# 亏损后的认知重建

## 核心目标
在经历连续亏损后，防止情绪化操作导致的"复仇交易"（Revenge Trading）。

## 核心观点
- 亏损是交易的一部分，不是失败，是成本
- 连续亏损往往是市场结构变化的信号，而非技术失误
- 在低置信度环境下，最好的操作是"不操作"

## 行为准则
1. **连续亏损 ≥ 3 次** → 暂停交易，复盘这 3 次的信号质量
2. **情绪激动时** → 切换至 HOLD，等待情绪平复（至少 30 分钟后）
3. **想要"追回来"时** → 这正是危险信号，必须降低 size 至常规的 50%

## 市场哲学
- 市场无法预测，但风险可以管理
- 好的交易是过程正确，不是结果正确
- 不确定时，轻仓；确定时，也要轻仓

## 信号过滤原则
- 在情绪状态（过于贪婪或恐惧）下产生的信号，置信度自动降低 50%
- 只交易你完全理解的信号，不熟悉的结构宁可放弃

## 输出映射
- 任何情绪化时机 → `{"decision": "HOLD", "size": 0, "reason": "emotional_state_detected"}`
- 正常状态 → 不干预策略信号
