1 — System Overview
Trait-Driven Architecture

ZeroClaw is built around 9 core traits that define every extension point. Implement a trait, register in the factory — the runtime handles the rest. No dynamic linking, no plugins, no scripting layer. Pure Rust all the way down.

Provider
LLM inference backends
trait Provider
Channel
Messaging platforms
trait Channel
Tool
Capability surface
trait Tool
Memory
Persistence backends
trait Memory
Observer
Telemetry & monitoring
trait Observer
RuntimeAdapter
Execution environments
trait RuntimeAdapter
Peripheral
Hardware boards (GPIO)
trait Peripheral
Sandbox
Isolated execution
trait Sandbox
Tunnel
Secure connectivity
trait Tunnel
