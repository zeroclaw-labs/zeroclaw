# Feature support matrix

This page compares ZeroClaw support against nearby Claw-family runtimes and deployment shapes. It is meant to answer "which claw supports this?" quickly, then point readers to the canonical docs for details.

The matrix is conservative. Hover a status emoji for the status word and row-specific explanation. A row marked <abbr title="Needs verification: this project needs a current source/docs pass before making a support claim">🔍</abbr> is not a current support claim.

## Deferred comparison work

This page intentionally does not claim row-by-row OpenClaw, PicoClaw, or NanoClaw-style parity. The matrix gives ZeroClaw-owned support claims and marks other runtime cells as <abbr title="Needs verification: this project needs a current source/docs pass before making a support claim">🔍</abbr> unless this repository has a current source-backed basis for a narrower statement.

Future updates should tighten individual cells only when they can link to canonical evidence. If a row cannot point to docs, an issue, an RFC/FND, or source-backed reference, keep it marked <abbr title="Needs verification: this project needs a current source/docs pass before making a support claim">🔍</abbr> instead of guessing.
