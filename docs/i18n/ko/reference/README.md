# 레퍼런스 카탈로그

## 핵심 레퍼런스

- 워크플로별 명령어: [cli/commands-reference.md](cli/commands-reference.md)
- Provider ID / 별칭 / 환경 변수: [api/providers-reference.md](api/providers-reference.md)
- Channel 설정 + allowlist: [api/channels-reference.md](api/channels-reference.md)
- Config 기본값 및 키: [api/config-reference.md](api/config-reference.md)

## 사용법

단계별 튜토리얼이 아닌, 정확한 CLI/config 세부 사항이나 provider 통합 패턴이 필요할 때 이 레퍼런스 모음을 활용하십시오.

새로운 레퍼런스/통합 문서를 추가할 때는 [../SUMMARY.md](../SUMMARY.md)와 [../maintainers/docs-inventory.md](../maintainers/docs-inventory.md) 양쪽에 반드시 링크를 추가해 주십시오.
