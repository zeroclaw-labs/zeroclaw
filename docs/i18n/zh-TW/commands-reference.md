## 適用場景

- 按工作查指令（onboard / status / doctor / channel 等）
- 對比指令引數與行為邊界
- 排查指令執行例外時確認預期輸出

## 使用建議

- 指令名、引數名、配置鍵保持英文。
- 行為細節以英文原文為準。

## 最近更新

- `zeroclaw gateway` 新增 `--new-pairing` 引數，可清空已配對 token 並在閘道器啟動時產生新的配對碼。
