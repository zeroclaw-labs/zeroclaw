# 官方管道與詐騙防範

此頁面是關於社群安全與冒充防禦的長期安全聲明。

## 詐騙警告

詐騙者可能會冒充 ZeroClaw 的維護者、貢獻者或社群成員。

如果有人自稱代表 ZeroClaw 並要求以下事項，請視為詐騙：

- 加密貨幣轉帳
- 錢包存取權或助記詞
- 私人財務資訊
- 官方安全通報流程以外的私人憑證

ZeroClaw 維護者不會透過私訊要求金錢或私人錢包/財務憑證。

## 官方資訊來源

請使用以下來源驗證公告：

- GitHub 儲存庫：`zeroclaw-labs/zeroclaw`
- GitHub 安全政策與公告：[../../SECURITY.md](../../SECURITY.md)

在 GitHub 儲存庫確認之前，請將第三方連結與社群媒體貼文視為不可信。

## 如何驗證公告

1. 確認相同的公告是否存在於 GitHub issues、PR、releases 或文件中。
2. 確認發布帳號是預期的專案維護者/組織帳號。
3. 優先使用來自儲存庫頁面的連結，而非轉發的私訊。

## 回報可疑活動

如果您發現冒充行為或詐騙外展：

1. 不要回應或發送資金/資料。
2. 擷取證據（螢幕截圖、使用者名稱、URL、時間戳記）。
3. 在 `zeroclaw-labs/zeroclaw` 開啟一個 GitHub issue，附上已脫敏的細節。

若需進行漏洞揭露，請使用私密通報：

- 安全政策：[../../SECURITY.md](../../SECURITY.md)
- 私密通報範本：[private-vulnerability-report-template.md](private-vulnerability-report-template.md)
