关于这套架构图

ZeroClaw是一个采用微内核架构设计的Rust AI代理运行时。这套可视化图表从四个关键维度展示了系统的架构设计:

  • 核心层次结构 - API traits定义 → Runtime核心 → 具体实现的三层架构
  • Crate组织 - 按稳定性层级(workspace crate tier)组织的模块依赖关系
  • 请求生命周期 - 从用户输入到响应输出的完整处理流程
  • 扩展点设计 - 六大核心trait及其实现方式

每个图表都聚焦于架构的一个特定方面，共同构成了对ZeroClaw系统设计的全面理解。建议使用现代浏览器查看这些SVG-based的交互式图表。
